üìã Overview

NotifyBot is a powerful Python script for automated email batch sending with two distinct operating modes:

üì® Single Mode

Sends ONE email to MULTIPLE recipients

  • Mass announcements
  • Company-wide updates
  • Newsletter distribution

üì¨ Multi Mode

Sends MULTIPLE personalized emails

  • Personalized reports
  • Department-specific updates
  • Customized notifications
üéØ Key Features: Dry-run testing, batch processing, email validation, signature support, image embedding, attachment handling, and comprehensive logging.

üöÄ Usage Commands

üß™ Testing (Dry Run)

‚ö†Ô∏è Always test first! Dry run sends draft emails only to approvers.

üéØ Live Execution

# Single mode with confirmation python notifybot.py --base-folder my-campaign # Multi mode, skip confirmation python notifybot.py --base-folder my-campaign --mode multi --force # Custom batch settings python notifybot.py --base-folder my-campaign --batch-size 100 --delay 3.0

üìä Command Options

Option Description Default
--base-folder Campaign folder name Required
--mode Force single/multi mode Auto-detect
--dry-run Test mode (approvers only) false
--force Skip confirmation false
--batch-size Emails per batch 500
--delay Seconds between batches 5.0

üîß Troubleshooting

‚ùå Common Issues

Error: Missing required files
Solution: Ensure all required files exist: subject.txt, body.html, from.txt, approver.txt
Error: Invalid base folder
Solution: Folder must exist inside /notifybot/basefolder/
Error: No recipients found (Single mode)
Solution: Provide at least one: to.txt, filter.txt+inventory.csv, additional_to.txt, cc.txt, or bcc.txt
Error: Multi mode requires filter.txt
Solution: Create filter.txt with filtering conditions for multi mode operation
Error: Sendmail not found
Solution: Install and configure sendmail on your system
